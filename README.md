Завершенное практическое задание на языке программирования Python. Задание состоит из двух частей: а) Часть 1 (файл main_part_one.py и пакет модулей my_package_part_one) формирует базу данных разработчиков ПО; б) Часть 2 (файл main_part_two.py и пакет модулей my_package_part_two) позволяет осуществить работу с базой данных (фильтровать, сортировать, формировать множества). Использованы модули Pandas, Numpy, Shutil
